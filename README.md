# Lithops UI
> Lightweight UI base on [lit](https://github.com/lit/lit)

## Preview Online
[https://cenfun.github.io/lithops-ui/](https://cenfun.github.io/lithops-ui/)
## Components
* Button
* Checkbox
* Flyover
* Input
* Loading
* Modal
* Progress
* Radio
* Select
* Skeleton
* Switch
* Tab
* Toast
* Upload
## Install
```sh
npm i lithops-ui
```
## Usage
```
import {
    LuiCheckbox,
    LuiInput
} from "lithops-ui";

```
